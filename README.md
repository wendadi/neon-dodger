# 霓虹躲避 (Neon Dodger) 🎮

一款为浏览器（完美适配手机与电脑）打造的高端、快节奏 H5 休闲游戏。非常适合用来在工作间隙放松。

## 🌟 游戏特色
- **零依赖:** 纯原生 HTML、CSS 和 JavaScript 编写，双击 `index.html` 即开即玩！
- **移动端优先:** 丝滑的拖拽滑屏手势，完美适配智能手机。
- **高级视效:** 令人惊艳的霓虹发光美学、毛玻璃 UI（Glassmorphism）、粒子爆炸视效以及由于 Canvas 加持的流畅运行。
- **渐进式难度:** 随着游戏进行会自动加快整体速度并密集生成障碍，极其考验反应。

## 🕹️ 怎么玩
- **操作:** 手机端按住屏幕左右拖动（电脑端鼠标拖拽）发出蓝色荧光的小球。
- **目标:** 不断躲避从上方掉落的红色星星。
- **得分:** 坚持不被撞击的时间越久，你的得分就越高！

## 🚀 启动指引（小白专属版）

这款游戏是专门为轻松娱乐设计，**100% 纯本地运行，绿色安全**：

1. **下载到本地**：在本项目 GitHub 页面上点击绿色大按钮 `Code` -> 选择 `Download ZIP` 下载。
2. **解压文件**：将下载后的压缩包解压到电脑的任意文件夹（例如您的桌面）。
3. **双击即开即可游玩**：找到里面的 `index.html` 文件，用鼠标 **双击** 打开它！（系统会自动用您的默认浏览器如 Chrome 或 Edge 帮您打开网页），您无需任何额外的按键，直接可以开始摸鱼了！

🌟 **为什么用本地下载玩更安全？**
- **完全断网可用**：这款游戏所有的代码（界面逻辑和特效）都已经在刚才那几 M 的文件里了，在游玩时它**完全不向外部联网和发送请求**，百分百隐私安全保护。
- **免除安全告警顾虑**：无需安装含有任何风险的 `.exe` 应用或安装乱七八糟的环境。因为浏览器的底层天然沙盒隔离运行机制，能彻底打消在办公电脑上游玩时被任何安全软件拦截的风险，您甚至可以把代码用记事本打开亲眼查验安全！

> *对于有一定技术基础的同学：如果你想将自己的电脑作为本地服务器开放给手机扫码/输入IP测试屏幕触摸，可以开个简易服务端（例如 `python3 -m http.server 8000` 等），并在同一局域网下访问电脑内部 IP 即可享受掌机的操作体验。*
